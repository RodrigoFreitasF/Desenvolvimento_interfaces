Elemento de imagem(img)

O elemento HTML que representa imagens é a tag <im>, essa tag é usada para exibir imagens. Para utiliza-la podemos usar imagens dos mais variados formatos,Para isso devemos usar os seguintes atributos: src, que representa o caminho da imagem e alt que repesenta um texto alternativo, que será carregado quando a imagem não estiver disponível.

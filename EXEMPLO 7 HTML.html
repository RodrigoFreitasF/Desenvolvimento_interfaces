Listas ordenadas <ol> e não ordenadas<ul>.

Este exemplo demonstra como criar listas ordenadase não ordenadas, influindo subitens(listas alinhadas). As listas são utilizadas para descrever de forma de bullets ou em ordem numérica ou de letras elementos.

Lista ordenada - <ol>

A lista ordenada é aquela que é definida por meio de números, letras ou algarísmos romanos.

A seguir demonstramos uma lista ordenada que usa as tags OL e LI, sendo LI o item da lista.

Exemplo 1 - lista ordenada

  1. Item 1
  2. Item 2
  3. item 3
  4. item 4
  5. Item 5

Exemplo 2 - lista ordenada

  1. Item 1
  2. Item 2
    1. Subiten2.1
    2. Subiten2.2
    3. Subiten2.3
    4. Subiten2.4
  3. item 3
  4. item 4
  5. Item 5
    1. Subiten5.1
    2. Subiten5.2
    3. Subiten5.3
    4. Subiten5.4

    6. Lista não ordenada - <ul>

    As listas não ordenadasSão aquelas onde os itens não aparecem com uma ordem específica. Essas listas utilizam marcadores como círculos e quadrados.

    As listas não ordenadas também são muito utilizadas na criação de menus.

    Exemplo de lista não ordenada

    • Item 1
    • Item 2
    • Item 3
      • SubItem 3.1
      • SubItem 3.2
      • SubItem 3.3
    • Item 4
    • Item 5

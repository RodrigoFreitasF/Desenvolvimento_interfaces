Exemplo 22 HTML CSS

CSS é o acrônimo para folhhas de estilo em cascata. Essa tecnica visa personalizar, estilizar um doucmento HTML. Para isso, usamos regras CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor: indica quais elementos serão afetados pelo estilo definido.
  • Bloco de declaração Neste caso o bloco é composto por propriedades e valores, estes elementos são aplicados aos elementos HTML

Seletores de Classe

O seletor de classe em CSS é aquele que é aplicado a um elemento HTML e que possui o atributo CLASS. Este seletor é utilizado quando queremos estilizar um documento HTML.

Este seletor sempre começará com um "." seguido pelo nome do seletor e suas propriedades e valores.

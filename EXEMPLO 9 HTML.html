Exemplo 9 HTML - Web Semântica e Links

A tag semântica <header> se refere ao cabeçalho do nosso documento HTML. Este elemento semântico é utilizado para definir uma seção introdutória, geralmente nele teremos tags de seção.

A tag semântica <nav> é usada para definir áreas de navegação dentro de uma página HTML. Ela geralmente contém menus de navegação. Abaixo exemplificamos um menu que será criado com a tag <a> representa um link de um documento qualquer. O elemento HTML main é uma tag semântica que define o conteúdo principal de uma página. Ela deve conter o conteúdo mais relevante para o usuário.

HTML semântico

O html semântico é conceito do html que se refere ao uso de elementos que possuem significado claro e descritivo, tanto para desenvolvedores quanto para navegadores. Além disso, eles são essenciais para os mecanismos de busca e leitores de tela.

Neste exemplo estamos utilizando a tag semântica section, essa tag agrupa contéudo relacionados dentro da tag main. Dentro da tag section poderemos ter também a tag article, essa tag representa um conteúdo independente, como posts de blog e notícias.

Tag de valor semântico neutro - <div>

A tag <div> é aquela que possui o valor semântico neutro. Ela é muito utilizada para agrupar elementos sem um significado específico. Geralmente a utilizamos quando queremos criar um layout.

tag <article>

Já a tag semântica <article> é utilizada para definir coteúdos independentes e auto suficientes.

Neste bloco estamos usando a tag aside, que é utilizada para representar conteúdos complementares ou secundários dentro de uma página. Nesta tag podemos ainda criar menus auxiliares e/ou anuncios diversos

Neste ultimo bloco estamos usando a tag semântica <footer> que é utilizada para representar o rodapé de uma página, ele geralmente traz informações sobre direitos autorais, informações de contato, créditos pessoais

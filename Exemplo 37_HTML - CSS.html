Exemplo 37 - Flexbox

O Flexbox é um dos sistemas mais importantes para criar layouts responsivos modernos. Ele permite que criemos layouts complexos de forma simples.

Para utiliza-lo precisamos usar a propriedade display com a opção flex. Neste exemplo usamos alguns recursos fundamentais da sua estrutura.

Neste exemplo mostraremos a utilização das propriedades Flexdirection que possui propriedades que permitem que um elemento possa ser organizado no sentido horizontal ou vertical.

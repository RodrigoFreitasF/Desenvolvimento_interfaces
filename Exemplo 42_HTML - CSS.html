O que é o grid Layout

O grid Layout é bidmensional, permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do Flexbox que é unidimensional, ordenando os elementos em linha.

Flexbox ou Grid Layout

O Flexbox e o CSS Grid Layout podem ser utilizados em conjunto. Você pode aplicar o flexbox dentro da grade criada pelo Grid Layout, aproveitando o melhor das duas tecnicas.

A escolha entre flexbox e Grid Layout não se trata de um substituir o outro, mas sim de entender as necessidades específicas do seu layout e escolher a tecnica que melhor atenda às suas necessidades.

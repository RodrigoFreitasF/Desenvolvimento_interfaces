Exemplo 35 CSS - Introdução a propriedade CSS Display

O que é a propriedade Display

A propriedade display no CSS especifica o comportamento de renderização de um elemento, ou seja, como ele é exibido na página e como ele interage com os elementos ao seu redor.

Principais valores da prpriedade Display

  • block: O elemento é exibido como um bloco, ocupando toda a largura disponível e iniciando em uma nova linha.
  • inline: O elemento é exibido na mesma linha, ocupando apenas o espaço necessário para seu conteúdo, sem iniciar uma nova linha.
  • inline-block: Combina características dos dois anteriores; o elemento é exibido na mesma linha, mas permite definir largura e altura.
  • flex: Define o elemento como um contêiner flexível, permitindo um layout mais dinâmico dos elementos filhos.
  • grid: Define o elemento como um contêiner de grade, facilitando a criação de layouts bidimensionais complexos.
  • none: O elemento não é exibido na página e não ocupa espaço.

Display: Block

Display: Block - O elemnto div com a classe .block é exibido como u bloco, iniciando em uma nova linha e ocupando toda a largura didsponível. A cor de fundo azul claro e a margem inferior de 10px ajudam a visualizar sua extensão.

Display: inline - block

Display inline - block os elementos div com a classe .inline-block são exibidos na mesma linha, mas permitem a definição de largura(150px) e altura(50px). A cor de fundo coral claro, a margem de 5pixels e o texto centralizado facilitam a vizualização de suas dimensões.

Exemplo 10 HTML - Elemento tabela

Para criarmos tabelas em documentos HTML devemos usar a tag <table>, essa tag permite que possamos organizar nossos dados em linhas e colunas . Cada tabela é composta por uma estrutura de linhas, cabeçalho e colunas(células).

Descrição das principais tags que são utilizadas para criar tabelas

  • <table> Define a tabela.
  • <tr>:Define a linha da tabela - tr é o acrônimo de table row.
  • <th>:Define o cabeçalho de uma coluna th(table header).
  • <td>:Define uma célula de dados (table data).

Exemplo 30 HTML - CSS Cores

Quando falamos de cores em CSS, estamos falando de uma infinidade de aplicações, que podem ser deifinidas de diversas formas. A seguir vamos descrever cada uma delas

RGB/RGBA

Essas cores são baseadas na combinação de cores obtidas a partir do vermelho, verde e Azul. Além disso podemos adicionar uma nova variante "a", o "a" se refere ao nível de transparencia, vai de zero a um. Qunato mais proximo do 1 for, menos transparente ele será.

Cores HEX

No caso das cores baseadas no sistema Hexadecimal as cores são formadas combinando o # com números que vão do 0 ao 9 e da letra "a" até o "f". O sistema Hexadecimal é muito comúm em documentos HMTL/CSS.

HSL/HSLA

O sistema de cores HSL é aquele qeu se baseia na MATIZ - HUE e é fornecido em graus no circulo cromático. Temos ainda o S de Saturação e o L de Luminosidade.

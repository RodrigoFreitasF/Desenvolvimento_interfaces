Exemplo 39 FlexBox / Flex-wrap

Flex-wrap: Wrap

A propriedade CSS Flex-wrap é aquela que define se os iten flexíveis são forçados a ficarem na mesma linha ou se podem ser quebrados em várias linhas.

O valor padrão do Flex-wrap é nowrap, o que significa que os itens flexíveis são forçados a ficarem na mesma linha, mesmo que isso signifique que eles seja redimensionados para caber no espaço disponível. Se o valor for Wrap, os itens flexíveis podem ser quebrados em várias linhas.

O Flex-wrap controla a direção em que as linhas são empilhadas. O valor Wrap padrão empilha as linhas da esquerda para a direita, se o Flex-direction for Row, ou de cima par baixo se ele for Column. O valor Wrap-reverse empilha as linhas da esquerda para a direita ou de baixo para cima, respectivamente.
